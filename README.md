## Script Monitoria de Qualidade
Script que vai pegar informações de cada atendimento realizado que estão em uma planilha, o nome dos agentes de atendimento de outra planilha e para cada agente vai aleatorizar uma interação para ser avaliado pelo monitor. </br>
Criei esse projeto para ajudar o time de qualidade a aleatorizar de forma automática os atendimentos para monitoria, pois o sistema de atendimento que usávamos não tinha essa opção, então o time estava fazendo de forma manual. 

### 📋 Pré-requisitos
Python mais recente, baixar em: https://www.python.org/downloads/ </br>
Bibliotecas: 
- requests
- pandas
### 🔧 Instalação
Para instalar as bibliotecas, use o código abaixo: </br>
`pip install requests` </br>
`pip install pandas` </br>

### 🛠️ Construído com
Utilizei o Visual Studio para desenvolver e rodar o código.
